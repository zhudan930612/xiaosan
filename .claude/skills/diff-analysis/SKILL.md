# 对比/差异分析任务强制工作流

## 触发条件

用户说：对比、区别、差异、核对、清单对照。

## 执行步骤

### 第一步：穷尽罗列源文档所有条目

- 将源文档（A文档）的所有条目逐条编号列出
- 保持原文，不概括、不合并、不省略

### 第二步：逐条在目标文档中查找对应

- 对源文档的每一条，在目标文档（B文档）中查找对应内容
- 标注结果：有 / 无 / 部分有
- 如果找不到对应，明确标注"无对应"

### 第三步：汇总成编号对照表

- 左边：源文档原条目编号和原文
- 右边：目标文档对应情况及标注
- 不允许合并行、不允许省略行

## 禁止行为

- 禁止凭经验筛选"重要"条目
- 禁止跳过看似"不相关"的条目
- 禁止合并多个条目为一行
- 禁止在未完成穷尽罗列前直接下结论
