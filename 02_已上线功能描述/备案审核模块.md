## 模块概述

### 业务场景

备案审核模块是监管端（街镇审核人员）对施工方/业主提交的备案申请进行审核管理的功能模块。审核人员通过该模块查看辖区内所有备案记录，对备案材料的完整性和合规性进行审核，决定通过或驳回，并跟踪备案的全生命周期状态。

**核心价值**：
- **街镇监管部门**：统一管理辖区内所有备案项目，提高审核效率，便捷标注问题字段
- **施工方/业主**：审核进度透明，驳回原因清晰，便于快速修改后重新提交

### 受众角色

#### 1. 街镇审核人员（监管端）
**角色定义**：负责审核备案申请的街镇监管部门工作人员

**使用场景**：
- 查看辖区内所有备案记录及审核进度
- 按工程类型和审核状态筛选备案
- 审核备案材料的完整性和合规性
- 对不符合要求的字段进行驳回并说明原因
- 审核通过后生成备案回执
- 删除待填报和已驳回的备案记录
- 查看审批历史记录

**典型用户**：
- 街镇建设管理部门工作人员
- 安全监管人员

**核心诉求**：
- 快速筛选待审核项目
- 清晰查看所有备案材料
- 便捷标注问题字段
- 高效完成审核工作

## 通用规则

### 1. 审核状态定义

| 列表页状态 | 说明 | 对应详情页状态 |
|-----------|------|--------------|
| 待填报 | 施工方已保存草稿但未提交备案申请 | 待填报 |
| 待审核 | 施工方已提交备案申请，等待街镇审核 | 待街道审核 |
| 已驳回 | 街镇审核不通过，需施工方修改后重新提交 | 街道审核驳回 |
| 已通过 | 街镇审核通过，已生成备案回执 | 街道审核通过 |

### 2. 列表排序规则

| 状态 | 排序字段 | 排序方向 |
|------|---------|---------|
| 待填报 | 发起备案时间 | 倒序 |
| 待审核 | 最新一次施工方提交备案申请时间 | 升序 |
| 已驳回 | 最新一次审核驳回时间 | 倒序 |
| 已通过 | 审核通过时间 | 倒序 |

### 3. 删除规则

- 仅"待填报"和"已驳回"状态的备案记录支持删除
- 删除操作需二次确认
- 删除后施工端同步删除对应的工程项目
- 删除后的项目进入回收站，保留15天
- 超过15天的项目从回收站彻底删除（前端页面删除，实际数据需在后台/数据库彻底删除）

### 4. 驳回规则

- 审核人员可同时驳回多个字段
- 每个驳回字段可单独输入驳回原因
- 驳回原因限200字
- 驳回记录按勾选的驳回选项逐条罗列
- 驳回格式示例：
  1. 施工合同
  2. 施工人员资质证书：证书已过期，请重新上传

## 功能描述

### 项目中心列表页

**入口路径**：内部政务平台(移动端) → 小型工程审核图标 → 项目中心页面

**页面名称**："项目中心"页面

**页面功能**：

**1. 顶部筛选区域**
- **筛选器**：显示"请选择"下拉框，用于选择筛选条件
- **搜索框**：提示文字"搜索工程名称或地址"，支持按工程名称或工程地址关键词搜索
- **搜索按钮**：点击进行搜索

**2. 工程类型Tab（横向切换）**
- 小型工程（显示该类型项目数量，如：小型工程(27)）
- 住宅装修（显示该类型项目数量，如：住宅装修(25)）
- 零星作业（显示该类型项目数量，如：零星作业(28)）
- 当前选中的Tab有绿色背景高亮标识
- Tab上红色圆点角标表示该类型下有新的待处理项目

**3. 审核状态筛选标签（横向排列）**
- 待填报（显示数量，如：待填报(6)）
- 待审核（显示数量，如：待审核(1)）
- 已驳回（显示数量，如：已驳回(15)）
- 已通过（显示数量，如：已通过(5)）
- 当前选中的状态标签有绿色文字和下划线标识
- "待审核"标签上有红色圆点角标表示有待审核项目

**4. 备案列表**
- 备案卡片列表展示，每个卡片包含：
  - 工程名称（加粗显示）
  - 工程地址（带定位图标，灰色文字）
- 列表按各状态指定的排序规则排列
- 列表底部显示"没有更多了"提示

**5. 左滑删除功能**
- 仅"待填报"和"已驳回"状态的备案卡片支持左滑删除
- 左滑后显示红色背景区域，带白色X图标和"删除"文字
- 点击"删除"按钮后弹出确认弹窗

**6. 删除确认弹窗**
- 弹窗顶部：铃铛图标 + "确定删除该申请记录?"
- 操作按钮：
  - "取消"按钮（白色背景，绿色边框和文字）：点击后关闭弹窗，不执行删除
  - "确定"按钮（绿色背景，白色文字）：点击后执行删除，备案记录进入回收站

**7. 回收站入口**
- 页面右下角显示"回收站"图标按钮（垃圾桶图标）
- 点击后进入回收站页面，查看已删除的工程项目
- 回收站中的项目保留15天，超过15天后无法恢复

**8. 底部导航**
- 项目中心（当前页面，绿色高亮，带建筑图标）
- 更多服务（灰色，带用户图标）

**业务规则**：
- 街镇审核人员可以查看辖区内所有备案记录
- 可以通过工程类型Tab切换查看不同类型的工程，切换后列表和数量重新加载
- 可以通过审核状态筛选标签筛选不同状态的备案
- 可以通过搜索框搜索工程名称或地址
- 各状态的列表按指定排序规则排列
- 仅"待填报"和"已驳回"状态的备案支持删除操作
- 删除操作会同步删除施工端对应的工程项目
- 删除后的项目进入回收站，保留15天，15天后彻底删除
- 点击备案卡片进入项目详情页面

### 项目详情页

**触发操作**：在[项目中心列表页](#项目中心列表页)点击任意备案卡片

**页面名称**："项目详情"页面

**页面功能**：

**1. 页面头部**
- 返回按钮（左上角）
- 页面标题："项目详情"

**2. 流程进度卡片（顶部区域）**
- 展示当前备案的审核流程状态，以双节点卡片形式呈现：
  - 左侧节点：历史状态（如"提交备案申请"），显示时间戳（如"03-19 15:48"），带绿色圆形图标
  - 中间：箭头连接线（>>>>>）
  - 右侧节点：当前状态（如"待街道审核"），带绿色圆形图标
- 右上角："审批记录"按钮，点击后弹出审批记录弹窗

**3. Tab栏（4个tab）**
- 基础信息
- 工程信息
- 安全信息
- 匹配保单
- Tab上可显示红色数字角标，表示该Tab下被驳回的字段数量

**4. 内容展示**
- 展示备案信息填写过程中的所有字段，以只读形式展示
- 字段以卡片列表形式组织，每个字段包含：
  - 左侧复选框（仅待审核状态下可勾选）
  - 字段名称（灰色文字）
  - 字段值（黑色文字）
- 被驳回的字段在列表中显示驳回原因提示

**5. 各状态详情页底部操作按钮差异**

| 状态 | 流程进度 | 底部操作按钮 |
|------|---------|------------|
| 待审核 | 提交备案申请 → 待街道审核 | "驳回"按钮（红色）+ "通过"按钮（绿色） |
| 已驳回 | 街道审核驳回 → 待施工单位重新填报 | "待施工单位填报"按钮（绿色渐变，仅展示） |
| 已通过 | 街道审核通过 → 已完成备案 | "查看回执"按钮（绿色渐变） |

**业务规则**：
- 不同状态的详情页展示不同的流程进度和底部操作按钮
- 待审核状态下，审核人员可以在字段左侧勾选复选框进行驳回选择
- 已驳回状态下，Tab上显示被驳回字段数量角标，被驳回的字段显示驳回原因
- 已通过状态下，可点击"查看回执"按钮查看备案回执

### 人工审核

**触发操作**：在[项目中心列表页](#项目中心列表页)点击"待审核"状态的备案卡片

**页面名称**："项目详情"页面（待审核状态）

**页面功能**：

**1. 驳回字段选择**
- 审核人员在Tab内容区点击字段左侧的复选框，勾选需要驳回的字段
- 可同时勾选多个字段进行批量驳回
- 已勾选的字段显示绿色对勾标识
- 底部显示"已选：X项"，可点击展开查看已选字段列表

**2. 驳回原因输入**
- 勾选字段后，在列表下方显示驳回原因输入区域
- 输入框提示文字："在此输入驳回原因（非必填）"
- 驳回原因限200字

**3. 驳回确认弹窗**
- 点击"驳回"按钮后，弹出驳回确认弹窗
- 弹窗顶部标题："驳回原因"，右侧有关闭按钮
- 弹窗内容：按序号罗列所有勾选的驳回字段名称（如：1. 作业场所  2. 工程总额(万)  3. 建筑面积(m²) 等）
- 每个字段右侧显示删除图标（垃圾桶），点击可从驳回列表中移除该字段
- 底部操作按钮：
  - "驳回"按钮（红色背景）：确认驳回，备案状态变更为"已驳回"

**4. 审核通过**
- 审核人员确认所有信息无误后，点击"通过"按钮
- 备案状态变更为"已通过"，工程正式纳入监管
- 系统自动生成备案回执

**5. 底部操作按钮**
- "驳回"按钮（红色背景，左侧）：点击后弹出驳回确认弹窗
- "通过"按钮（绿色背景，右侧）：点击后直接通过审核

**业务规则**：
- 人工审核由街道级审核人员进行
- 审核人员可以点击任意字段左侧的复选框进行驳回选择，可同时驳回多个字段
- 驳回原因输入框为非必填，但建议填写以便施工方了解驳回原因
- 驳回记录按勾选的驳回选项逐条罗列，格式示例：
  1. 工程名称：test-tp01-工程名称
  2. 工程类型：test-tp02-工程类型
  3. 是否涉及用途转换装修工程：test-tp03-是否涉及用途转换装修工程
- 点击"通过"按钮：备案状态变更为"已通过"，进入日常巡查模块
- 点击"驳回"按钮：备案状态变更为"已驳回"，施工方需要修改被驳回的字段后重新提交
- 审核操作（通过/驳回）会记录在审批记录中
- 驳回后，施工方可在施工端查看驳回详情并修改后重新提交备案申请

### 审批记录

**触发操作**：在项目详情页点击右上角"审批记录"按钮

**展示形式**：底部弹窗（从底部滑出）

**页面功能**：

**1. 弹窗头部**
- 标题："审批记录"
- 右上角关闭按钮（X图标）

**2. 审批记录列表**
- 按时间倒序显示所有审批记录，最新的记录在最上方
- 每条记录以时间线形式展示：
  - 左侧：状态节点图标（空心圆表示历史节点，实心圆表示当前节点）
  - 时间戳：如"03-19 15:48"
  - 审批节点名称：如"待街道审核"、"提交备案申请"、"街道审核驳回"
  - 审批人信息：如"何夏梅 | 185*****743"
- 驳回记录会详细展示驳回原因列表：
  - 驳回原因按序号罗列
  - 每条驳回原因包含字段名称和具体原因
  - 右侧显示"已驳回"红色印章标识

**3. 审批记录状态样式**

| 状态节点 | 图标样式 | 文字说明 |
|---------|---------|---------|
| 发起备案 | 蓝色圆形+白色加号 | 显示发起人和时间 |
| 等待施工单位填报 | 蓝色圆形+白色时钟 | 保存了基础信息，工程信息及安全信息匹配保单还没填写完，没提交备案申请前 |
| 提交备案申请 | 蓝色圆形+白色文档 | 仅一种情况：施工单位提交备案申请，显示姓名和手机号 |
| 等待审核 | 蓝色圆形+白色时钟 | 需要谁审核，就显示【xx审核】，包括：待街道审核、ai审核中 |
| 人工审核驳回 | 红色圆形+白色叉号 | 谁驳回的就显示【xx审核驳回】，显示姓名+组织和手机号 |
| AI审核驳回 | 红色圆形+白色叉号 | 分为当前状态展示及申请记录的详细展示，显示需修改内容 |
| 等待修改 | 蓝色圆形+白色时钟 | 驳回后，需要施工方重新修改，施工方未提交备案申请的状态 |
| 人工审核通过 | 绿色圆形+白色对勾 | 谁通过的，就显示【xx审核通过】，显示姓名+组织和手机号 |
| AI审核通过 | 绿色圆形+白色对勾 | 分为当前状态展示及申请记录的详细展示 |
| 备案完成 | 绿色圆形+白色对勾 | 审核通过，生成备案回执 |

**业务规则**：
- 审批记录按时间倒序排列
- 每个审批节点都会记录审批时间、审批人、审批状态、审批意见
- 用户可以查看完整的审批流程历史
- 审批记录页面为只读展示，不支持任何编辑操作

### 删除与回收站

**删除功能**

**触发操作**：在项目中心列表页，对"待填报"或"已驳回"状态的备案卡片左滑，点击"删除"按钮

**页面功能**：
- 左滑删除：仅"待填报"和"已驳回"状态的备案卡片支持左滑删除
- 删除确认：点击删除后弹出确认弹窗，需二次确认
- 同步删除：删除后施工端对应的工程项目同步删除
- 进入回收站：删除后的项目进入回收站，保留15天

**回收站功能**

**入口**：项目中心列表页右下角"回收站"图标按钮

**页面功能**：
- 展示已删除的备案记录列表
- 支持恢复操作（15天内）
- 超过15天的项目无法恢复，提示"已过恢复期"
- 回收站中的项目保留15天后从后台/数据库彻底删除

**业务规则**：
- 仅"待填报"和"已驳回"状态的备案支持删除操作
- 删除操作需二次确认，防止误删
- 删除后施工端同步删除对应的工程项目
- 删除后的项目进入回收站，保留15天
- 15天内可从回收站恢复已删除的项目
- 超过15天的项目从前端页面和后台数据库彻底删除
- 回收站中的项目显示删除时间和剩余恢复天数

## 审批记录状态说明

以下为审批记录中各状态节点的详细说明：

| 状态节点 | 样式 | 说明 |
|---------|------|------|
| 发起备案 | 蓝色加号图标 + "施工单位发起备案" + 发起人信息 | 谁发起的备案，就显示【xx发起备案】 |
| 等待施工单位填报 | 蓝色时钟图标 + "待施工单位填报" | 保存了基础信息，工程信息及安全信息匹配保单还没填写完，没提交备案申请前 |
| 提交备案申请 | 蓝色文档图标 + "施工单位提交备案申请" + 姓名和手机号 | 仅一种情况：施工单位提交备案申请 |
| 等待审核 | 蓝色时钟图标 + "待街道审核" | 需要谁审核，就显示【xx审核】，包括：待街道审核、ai审核中 |
| 人工审核驳回 | 红色叉号图标 + "街道审核驳回" + 驳回原因列表 | 谁驳回的就显示【xx审核驳回】：街道审核驳回 |
| AI审核驳回 | 红色叉号图标 + "AI审核驳回" + 需修改内容列表 | 分为当前状态展示及申请记录的详细展示 |
| 等待修改 | 蓝色时钟图标 + "待施工单位重新填报" | 驳回后，需要施工方重新修改，施工方未提交备案申请的状态 |
| 人工审核通过 | 绿色对勾图标 + "物业审核通过" + 通过人信息 | 谁通过的，就显示【xx审核通过】：街道审核通过 |
| AI审核通过 | 绿色对勾图标 + "AI审核通过" + AI员工信息 | 分为当前状态展示及申请记录的详细展示 |
| 备案完成 | 绿色对勾图标 + "已完成备案" | 审核通过，生成备案回执 |
