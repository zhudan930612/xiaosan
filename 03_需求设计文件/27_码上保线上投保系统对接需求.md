## 一、文档目的

本文档由平台侧输出，用于统一青浦区小型工程安全生产责任保险的标准对接口径，供太保、平安、国任等保险公司产品、开发、联调人员使用。

本文档只回答 4 类核心问题：

1. 业务流程怎么走。
2. 平台传给保司什么数据。
3. 保司回传给平台什么结果。
4. 双方各自负责什么、还需确认什么。

---

## 二、标准业务流程

受平台能力边界约束，备案平台内不支持直接嵌入交易性行为，因此本期采用“平台展示保司入口 + 保司短信下发投保链接 + 浏览器完成支付 + 保单结果回传平台”的方案。

标准流程如下：

1. 备案平台展示可选的线上投保保险公司。
2. 用户在备案流程中选择某一家保险公司。
3. 平台向保险公司传递基础投保信息。
4. 保险公司生成订单并向用户发送投保短信。
5. 用户点击短信链接，在浏览器打开保险公司投保页面并完成支付。
6. 保险公司将电子保单等结果信息回传至备案平台。
7. 平台接收保单并关联备案。

流程约束如下：

- 备案平台负责展示入口和承接结果，不承载交易支付页面。
- 实际支付发生在保险公司侧页面，不发生在备案平台内。
- 短信链接是本期进入保险公司投保页的主要承接方式。

```mermaid
    E --> F[浏览器打开保司投保页并支付]
    F --> G[保险公司回传保单结果]
    G --> H[平台接收保单并关联备案]
```

## 三、对接范围

### 3.1 本期纳入范围

| 事项 | 说明 |
|------|------|
| 投保入口拉起 | 平台在随申办三端展示并发起线上投保流程 |
| 投保短信 | 保险公司向用户发送订单短信和投保链接 |
| 保单结果回传 | 保险公司向平台回传投保结果和电子保单信息 |
| 电子保单推送 | 投保成功后向用户邮箱发送电子保单 |
| 电子发票推送 | 投保成功后按投保人口径开票并推送 |

### 3.2 本期不纳入范围

| 事项 | 处理方式 |
|------|---------|
| 退保 | 线下处理 |
| 批改 | 线下处理 |
| 对账 | 线下处理 |
| 理赔 | 线下处理 |

## 四、平台传保司字段清单

以下字段为平台当前标准输出字段，后续太保、平安、国任对接时原则上优先复用。

### 4.1 投/被保险人信息

| 字段 | 字段类型 |
|------|---------|
| 投/被保险人名称 | 文本 |
| 统一社会信用代码号 | 数值 |
| 备案人姓名 | 文本 |
| 备案人手机号码 | 数值 |

### 4.2 工程信息

| 字段 | 字段类型 |
|------|---------|
| 工程名称 | 文本 |
| 工程类型 | 文本 |
| 工程地址 | 文本 |
| 所属街镇 | 文本 |
| 主要施工内容 | 文本 |
| 施工负责人 | 文本 |
| 施工负责人手机号码 | 数值 |
| 工程造价 | 数值 |
| 工程起期 | 日期 |
| 工程止期 | 日期 |

### 4.3 附件资料

| 字段 | 字段类型 |
|------|---------|
| 施工单位营业执照 | JPG 或其他图片格式 |
| 承发包合同 | JPG、PDF 或其他格式 |

### 4.4 字段说明

- 上述为平台业务字段，不要求与各保司最终接口字段名完全一致。
- 各保司如字段命名、类型定义、附件传输方式不同，应在接口适配层完成映射。
- 统一社会信用代码号、手机号当前资料口径写为“数值”，最终是否按字符串传输，以保司接口文档为准。
- 附件资料的传输方式需进一步确认是文件流、URL、Base64 还是其他方式。

## 五、保司回传平台字段要求

### 5.1 最小回传字段集

| 字段 | 说明 |
|------|------|
| 交易流水号 | 与平台发起投保请求一一对应 |
| 保单号 | 投保成功后生成的正式保单号 |
| 投保状态 | 本期至少明确“投保成功”状态 |
| 电子保单 URL | 用于查看或下载电子保单 |

### 5.2 回传要求

- 回传时机：用户支付成功后回传。
- 重试次数：最多 3 次。
- 重试时限：无时间限制。
- 平台响应：平台需返回明确成功/失败结果，便于保险公司判断是否重试。

## 六、双方职责与交付物

### 6.1 平台侧职责

- 展示线上投保保险公司入口。
- 按约定规则传递投保参数。
- 提供保单结果回传接口。
- 接收回传结果并关联备案业务。

### 6.2 保险侧职责

- 提供投保链接拉起能力及接口文档。
- 生成订单并向用户发送投保短信。
- 承接投保页面、产品展示、信息确认和支付。
- 投保成功后发送电子保单、电子发票，并向平台回传结果。

### 6.3 双方交付物

| 提供方 | 交付物 |
|--------|--------|
| 保险侧 | 带参拉起投保链接接口文档、字段映射附件、联调环境说明 |
| 平台侧 | 保单状态回传接口文档、平台字段清单、联调配合信息 |
