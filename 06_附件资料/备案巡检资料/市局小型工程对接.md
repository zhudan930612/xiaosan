数据对接文档
V1.0
一、对接内容和流程
上海市小型房建工程信息系统（以下简称本系统）提供小型项目事项对接接口。各区业务系统小型项目事项办理完成后将数据归集到本系统。
二、接口标准（备案登记事项）
2.1获取安全令牌
说明：获取安全令牌
接口地址：提供中台服务
请求方式：post
请求格式：String Body（application/json）

请求具体参数说明
字段名	类型	必需	说明
account	String	是	账号(单独提供)
sign	String	是	签名值MD5(account+secretkey)
返回具体参数说明
返回实例
{
    "success":true,
"message":"成功",
"result":{
"access_token":"d83bb786b0c53ab882dc809071a73a4c",
"expires_in": 3600
}
}

返回具体参数说明
字段名	类型	必需	说明
success	boolean	是	成功状态(true=成功 false=失败)
message	String	是	返回消息
result	Object	是	返回结果
access_token	String	是	安全令牌
expires_in	Int	是	有效期，单位s
