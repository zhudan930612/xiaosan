# 巡检与备案报表指标体系参考

> 本文档汇总小型工程安全纳管系统中备案管理和日常巡查相关的报表指标，供报表设计和周报/月报模板设计参考。
> 文档状态：参考材料，非最终设计依据。

---

## 一、报表使用角色

| 角色 | 报表需求 | 关注重点 |
|------|---------|---------|
| **区级管理人员** | 全局统计、街镇对比、考核上报 | 全区态势、各街镇排名、年度考核数据 |
| **街镇管理人员** | 辖区明细、工作台账、整改跟踪 | 本辖区备案/巡查/隐患闭环情况 |
| **网格员/安全协管员** | 个人巡查记录、待办清单 | 自己的巡查任务、发现的隐患状态 |
| **区领导/市级平台** | 汇总报表、数据上报 | 宏观数据、政策执行效果 |
| **保险公司** | 事故预防服务统计 | 服务覆盖率、隐患整改率 |

## 二、报表形式

| 形式 | 适用场景 | 特点 |
|------|---------|------|
| **PC管理后台报表** | 日常查询、明细导出 | 可筛选、排序、分页，支持导出Excel |
| **智慧大屏可视化** | 实时展示、领导视察 | 地图热力图、趋势图表、排名榜单 |
| **移动端报表** | 现场查看、个人统计 | 巡查员个人报表、待办提醒 |
| **PDF汇报材料** | 正式汇报、存档 | 固定排版，适合打印和传阅 |
| **Excel导出** | 二次分析、数据上报 | 原始数据，支持透视分析 |

## 三、备案相关指标

### 3.1 备案数量类

| 指标名称 | 计算口径 | 维度拆分 |
|---------|---------|---------|
| 总备案数量 | 历史累计已通过审核的备案数 | 按工程类型/街镇/时间 |
| 新增备案数量 | 周期内新提交的备案数 | 今日/本周/本月/本年 |
| 待审核备案数 | 当前待审核状态的备案数 | 按街镇/工程类型 |
| 审核通过数 | 周期内审核通过的备案数 | 按审核方式（AI/人工）|
| 审核驳回数 | 周期内被驳回的备案数 | 按驳回原因 |
| 驳回率 | 驳回数 / 提交总数 x 100% | 按街镇/工程类型 |
| 平均审核时长 | 从提交到审核通过的平均时间 | 按街镇/审核方式 |
| 高风险工程占比 | 高风险工程数 / 总备案数 x 100% | 按街镇/工程类型 |

### 3.2 备案质量类

| 指标名称 | 说明 |
|---------|------|
| 材料完整率 | 一次性审核通过的备案占比 |
| AI审核通过率 | AI审核通过数 / AI审核总数 |
| 人工复核率 | AI通过后仍需人工审核的占比 |
| 重复提交率 | 被驳回后重新提交的平均次数 |
| 逾期备案率 | 超过规定时限才备案的工程占比 |

### 3.3 备案结构类

| 指标名称 | 维度 |
|---------|------|
| 工程类型分布 | 小型工程/住宅装修/零星作业占比 |
| 风险类型分布 | 动火/高空/动土/用电等占比 |
| 街镇分布 | 各街镇备案数量及占比 |
| 施工单位集中度 | TOP10施工单位承接工程数 |
| 投资规模分布 | 按投资额区间统计 |

## 四、巡检（巡查）相关指标

### 4.1 巡查覆盖类

| 指标名称 | 计算口径 |
|---------|---------|
| 巡查覆盖率 | 实际巡查工程数 / 应巡查工程数 x 100% |
| 应巡查次数 | 按频次规则计算的理论巡查次数 |
| 实际巡查次数 | 周期内实际完成的巡查次数 |
| 巡查完成率 | 实际巡查次数 / 应巡查次数 x 100% |
| 人均巡查量 | 巡查次数 / 巡查人员数 |

> 应巡查次数计算需结合三方频次规则：
> - 街道：高风险每10天1次，非高风险住宅每30天1次，其他每21天1次
> - 社区：高风险每14天1次，非高风险住宅每21天1次，其他每14天1次
> - 物业：高风险每2天1次，非高风险住宅每7天1次，其他每3天1次

### 4.2 隐患发现与整改类（核心）

| 指标名称 | 计算口径 |
|---------|---------|
| 隐患发现数 | 周期内巡查发现的隐患总数 |
| 隐患发现率 | 发现隐患的工程数 / 巡查工程数 x 100% |
| 隐患整改率 | 已整改隐患数 / 发现隐患总数 x 100% |
| 隐患按期整改率 | 按期整改隐患数 / 应整改隐患总数 x 100% |
| 隐患逾期率 | 逾期未整改隐患数 / 应整改隐患总数 x 100% |
| 平均整改周期 | 从发现到整改完成的平均天数 |
| 停工整改率 | 停工整改数 / 总隐患数 x 100% |
| 现场整改率 | 现场整改数 / 总隐患数 x 100% |
| 移交执法率 | 移交执法部门数 / 总违规数 x 100% |

### 4.3 隐患分类统计

| 维度 | 说明 |
|------|------|
| 隐患等级分布 | 较大隐患 / 一般隐患占比 |
| 隐患类型分布 | 按标准化隐患清单分类统计 |
| 隐患场景分布 | 人为操作/机械设备/管理问题/工作环境 |
| 高发隐患TOP10 | 出现频次最高的隐患类型 |

### 4.4 巡查人员绩效类

| 指标名称 | 适用对象 |
|---------|---------|
| 个人巡查次数 | 网格员/协管员 |
| 个人隐患发现数 | 网格员/协管员 |
| 整改跟踪完成率 | 网格员/协管员 |
| 平均巡查时长 | 单次巡查平均用时 |
| 问题上报及时率 | 24小时内上报率 |

## 五、综合类指标

### 5.1 工程全生命周期

| 指标名称 | 说明 |
|---------|------|
| 在库工程数 | 已备案通过且未完工销项的工程数 |
| 工程状态分布 | 正常施工/现场整改/停工整改/已核销占比 |
| 平均施工周期 | 从备案通过到完工销项的平均天数 |
| 逾期未销项率 | 超期未办理销项的工程占比 |

### 5.2 "三张清单"兼容指标

| 清单 | 对应指标 |
|------|---------|
| **底数清单** | 辖区工程总数、按类型/风险/街镇分布的底数 |
| **隐患清单** | 未整改隐患明细、逾期隐患预警、隐患分类统计 |
| **整改清单** | 整改中隐患跟踪、已整改隐患销号、责任人到岗情况 |

### 5.3 考核类指标

| 指标名称 | 说明 |
|---------|------|
| 备案受理及时率 | 1个工作日内完成审核的占比 |
| 巡查覆盖率 | 实际巡查 / 应巡查 |
| 隐患整改率 | 已整改 / 发现总数 |
| 系统使用率 | 登录频次、数据录入完整度 |
| 街镇综合排名 | 多指标加权评分 |

## 六、指标使用建议

### 6.1 按报表类型选择指标

| 报表类型 | 推荐指标 |
|---------|---------|
| **实时统揽** | 新增备案数、待审核数、巡查次数、隐患发现数 |
| **周报** | 本周新增/审核/巡查/隐患数、环比变化、覆盖率、整改率 |
| **月报** | 月度累计数、趋势分析、街镇排名、目标完成度、风险预警 |
| **考核报表** | 备案及时率、巡查覆盖率、隐患整改率、逾期率、综合排名 |
| **专项报告** | 按主题选取关联指标，如住宅装修专项侧重备案结构+隐患类型 |

### 6.2 指标计算注意事项

1. **时间口径一致**：同一报表中的指标时间范围需统一（提交时间 vs 审核时间 vs 巡查时间）
2. **去重规则明确**：同一工程多次备案按最新有效备案计算
3. **环比基准清晰**：本周环比上周，本月环比上月
4. **分母不为零**：计算比率时分母为0时显示"--"
5. **三方巡查分开统计**：街道/社区/物业巡查数据需独立计算和展示
