管理后台操作手册：组织与用户权限配置
========================================

本文档用于指导运营人员/系统管理员在青浦区小散工程安全监管平台管理后台中完成监管组织的权限配置和用户的数据权限开通。


一、整体流程概述
----------------

1.1 开通的两条线

监管人员要使用系统，必须同时完成以下两条线的开通，缺一不可：

    线A - 数据局侧（入口权限）：
        由数据局在政务云平台开通，给用户分配备案审核PC端、移动端、智慧大屏的入口图标。
        没有入口权限，用户登录政务云后看不到系统入口。

    线B - 系统侧（数据权限）：
        由系统管理员在管理后台配置，给用户分配可审核的工程类型、作业场所、所属组织、角色等。
        没有数据权限，用户即使能看到入口，也看不到任何备案数据。

只有线A和线B都完成，用户才能真正使用系统。


1.2 组织和用户的来源

    组织：由数据局同步到本系统，不是直接在管理后台创建的。
    用户：由数据局同步到本系统，不是直接在管理后台创建的。

管理后台的操作，是对数据局已同步过来的组织和用户，进行系统内的数据权限配置（工程类型、作业场所、角色分配等）。


1.3 数据访问范围限制

数据局目前只给部分单位开通了数据同步权限，当前已开通的单位如下：

    街镇/园区：
        青浦工业园区、赵巷镇、华新镇、重固镇、白鹤镇、朱家角镇、练塘镇、金泽镇、盈浦街道、香花桥街道、西虹桥公司、徐泾镇、夏阳街道

    区级部门：
        区建管委、区市场监管局、区国资委、区房管局、区机管局、区教育局、区卫健委、区民政局、区绿化市容局、区商务委、区体育局、区应急局、区文旅局、区数据局（区信息委、区政务办）、区规划资源局、区水务局、区农业农村委、区国动办

    其他：
        小型工程系统筹备组

如果要新增的单位不在以上列表中，必须先联系数据局申请开通该单位的数据同步权限，否则数据局不会把该单位的组织和用户同步到本系统。


二、新增单位/组织的完整流程
----------------------------

2.1 判断该单位是否已开通数据同步

步骤：
1. 查看本章 1.3 节的数据访问范围列表
2. 确认目标单位是否在列表中

情况A：单位已在列表中
    → 直接进入第三章"配置组织和用户权限"

情况B：单位不在列表中
    → 按 2.2 节流程，先找数据局申请开通


2.2 单位不在列表中：申请开通数据同步

适用场景：目标单位尚未在数据局的数据访问范围内

操作步骤：
1. 联系数据局，申请将该单位纳入"小型工程系统"的数据访问范围
2. 申请内容需明确：
    - 单位名称
    - 需要同步的数据类型（组织、用户）
3. 等待数据局审批并完成配置
4. 数据局配置完成后，该单位的组织和用户会自动同步到本系统
5. 同步完成后，进入第三章"配置组织和用户权限"

注意事项：
    - 数据局开通权限需要审批流程，耗时视具体情况而定，需提前申请
    - 只有数据局完成开通后，该单位的组织和用户才会出现在管理后台


三、配置组织和用户权限
----------------------

3.1 权限配置的前提条件

开始配置前，请确认：

    □ 该单位已由数据局开通数据同步权限
    □ 该单位的组织已同步到管理后台（企业中心 → 监管单位 中可见）
    □ 该单位的用户已同步到管理后台（运营管理 → 用户管理 → 监管方 Tab 中可见）

如果以上任何一项不满足，请先返回第二章处理。


3.2 给组织配置数据权限（工程类型 + 作业场所）

操作步骤：
1. 登录管理后台
2. 进入 企业中心 → 监管单位
3. 在列表中找到目标组织
4. 点击该组织所在行的【编辑】按钮
5. 配置以下数据权限：

    小型工程类型：勾选该组织可审核的工程类型（可多选）
    零星作业类型：勾选该组织可审核的零星作业类型（可多选）
    住宅装修类型：勾选该组织可审核的住宅装修类型（可多选）
    小型工程作业场所：勾选该组织可管理的作业场所（可多选）
    零星作业作业场所：勾选该组织可管理的作业场所（可多选）

6. 点击【保存】

可选工程类型清单：

    小型工程类型（12类）：
        新改扩房屋建设工程
        交通工程
        水务工程
        园林绿化工程
        既有建筑修缮工程
        建筑装饰装修工程
        既有建筑加装电梯工程
        临时建设工程
        农民自建房工程
        农业设施建设工程
        农村地区综合性建设工程
        人防工程

    住宅装修类型（1类）：
        竣工验收合格后的住宅室内装饰装修

    零星作业类型（10类）：
        空调、太阳能、雨棚、防盗网、管道、工业环保设备设施等非主体工程配套设备设施的安装、维护、拆除作业
        建筑外墙清洗、修补、屋面检修等各类建筑外墙零星高处作业
        户外小型广告及门楣招牌的安装、维护、拆除作业
        新能源汽车、电动自行车充电设施的安装、维护、拆除作业
        临时搭建舞台、看台、展台、布景等搭设、拆除作业
        锅炉、压力容器（含气瓶）、压力管道、电梯、起重机械等特种设备的安装、改造、修理
        住宅区、公共场所绿化枝叶修剪
        道路绿化枝叶修剪
        供水、排水、通信、电力、燃气等管线设施维护、维修工程
        其他由存在坍塌、物体打击、高处坠落、消防安全、机械伤害等事故风险的零星作业活动

可选作业场所清单（12项）：
    商业综合体、大型商场、超市、商品交易市场等商贸场所
    工业企业内场所
    医疗卫生机构内场所
    学校、幼儿园等教育机构内场所
    文化场馆、旅游景区场所
    体育场场所
    党政机关集中办公区内场所
    养老机构、福利院等民政服务机构内场所
    区属国有企业经营管理场所内场所
    住宅小区公共区域内场所
    区管交通设施（如客运站、停车场）范围内场所
    涉及公共绿地、公园、环卫设施等场所


3.3 给用户配置数据权限（角色 + 权限模式）

适用场景：数据局已同步过来的用户，需要配置系统内的数据权限

操作步骤：
1. 登录管理后台
2. 进入 运营管理 → 用户管理
3. 切换到【监管方】Tab
4. 在列表中找到目标用户（可按手机号或姓名搜索）
5. 点击该用户所在行的【编辑】按钮
6. 配置以下信息：

    所属监管单位（必填）：下拉选择已同步过来的组织
    角色（必填）：单选
        - 普通审核员：按工程类型和作业场所过滤可见备案
        - 科室负责人：可见本科室全部备案
        - 街道领导：可见本街镇/园区全部备案
        - 区级领导：可见全区所有备案
    权限模式（必填）：单选
        - 继承：用户权限跟随所属组织的配置自动更新（推荐）
        - 自定义：用户拥有独立的权限配置

7. 如权限模式选择【自定义】，需额外勾选：
    - 可审核的小型工程类型
    - 可审核的住宅装修类型
    - 可审核的零星作业类型
    - 可管理的小型工程作业场所
    - 可管理的零星作业作业场所

8. 点击【保存】

建议：90% 以上的用户应采用"继承"模式。仅当某个人员的职责明显不同于其所在科室的常规职责时，才使用"自定义"模式。


3.4 批量配置权限（Excel 导入）

适用场景：需要同时给多个用户配置权限（3人及以上）

操作步骤：
1. 打开人员导入模板（参照《导入模板_最终版2.xlsx → 人员导入》）
2. 按模板要求填写数据
3. 进入 运营管理 → 用户管理 → 监管方 Tab
4. 点击页面右上角【导入】按钮
5. 上传填写好的 Excel 文件
6. 确认导入结果

模板字段说明：

    姓名（必填）：用户真实姓名
    手机号（必填）：用户手机号，用于登录
    所属组织名称（必填）：必须与后台已同步的组织名称完全一致
    角色（必填）：单选，普通审核员 / 科室负责人 / 街道领导 / 区级领导
    权限模式（必填）：单选，继承 / 自定义
    自定义小型工程类型（选填）：权限模式为"自定义"时填写
    自定义住宅装修类型（选填）：同上
    自定义小型工程作业场所（选填）：同上
    自定义零星作业场所（选填）：同上

多选字段填写规则：以中文逗号（，）分隔，可填"全部"表示全部选中，以"非"开头表示排除。


四、给用户开通政务云入口权限
------------------------------

4.1 整理名单并提交数据局

操作步骤：
1. 按以下格式整理需要开通入口权限的人员名单：

    姓名：用户真实姓名
    手机号：用户在政务云平台注册的手机号
    所属组织：该用户在政务云平台所属的单位/组织名称

2. 将整理好的名单发送给数据局
3. 明确告知数据局需要开通的入口：
    - 备案审核 PC 端
    - 备案审核移动端
    - 智慧大屏
4. 等待数据局完成配置


4.2 验证入口权限是否开通

数据局配置完成后，验证方法：
1. 让用户登录政务云平台
2. 检查用户是否能在政务云首页看到"小型工程系统"的入口图标
3. 如能看到入口图标，说明线A（入口权限）已开通
4. 点击进入系统，检查是否能正常查看备案数据
5. 如能看到数据，说明线B（数据权限）也已配置正确


五、完整开通流程图
------------------

新增单位的完整流程：

    步骤1：确认单位是否在数据局数据访问范围内
        ├── 是 → 直接进入步骤3
        └── 否 → 步骤2

    步骤2：联系数据局，申请将该单位纳入数据访问范围
        └── 等待数据局审批并完成配置

    步骤3：确认组织和用户已同步到管理后台
        ├── 组织可见（企业中心 → 监管单位）
        └── 用户可见（运营管理 → 用户管理 → 监管方）

    步骤4：在管理后台给组织配置数据权限
        └── 工程类型、作业场所

    步骤5：在管理后台给用户配置数据权限
        └── 所属组织、角色、权限模式

    步骤6：整理名单，提交数据局开通政务云入口
        └── 姓名、手机号、所属组织

    步骤7：等待数据局开通入口权限

    步骤8：验证用户是否可正常使用系统
        └── 能看到入口 + 能看到数据 = 开通成功


已有单位新增用户的完整流程：

    步骤1：确认用户是否已由数据局同步到管理后台
        ├── 已同步 → 直接进入步骤3
        └── 未同步 → 步骤2

    步骤2：联系数据局，确认该用户已在政务云平台注册，并同步到本系统
        └── 等待同步完成

    步骤3：在管理后台给用户配置数据权限
        └── 所属组织、角色、权限模式

    步骤4：整理名单，提交数据局开通政务云入口
        └── 姓名、手机号、所属组织

    步骤5：等待数据局开通入口权限

    步骤6：验证用户是否可正常使用系统


六、常见问题排查
----------------

6.1 用户登录后看不到系统入口

原因：数据局尚未开通该用户的政务云入口权限（线A未完成）
解决：联系数据局，确认该用户的入口权限已开通


6.2 用户能看到入口，但看不到任何备案数据

原因：管理后台未给该用户配置数据权限（线B未完成）
解决：进入 运营管理 → 用户管理 → 监管方，检查该用户的所属组织、角色、权限模式是否已正确配置


6.3 用户能看到部分备案，但不是全部

原因：该用户的权限模式为"继承"，但所属组织的工程类型配置不完整；或权限模式为"自定义"，自定义范围过小
解决：
    - 如应为继承模式：检查所属组织的工程类型和作业场所配置是否完整
    - 如应为自定义模式：检查该用户的自定义权限范围是否正确


6.4 组织列表中找不到目标单位

原因1：该单位尚未由数据局同步到本系统
解决：确认该单位已在数据局的数据访问范围内，如不在则先申请开通

原因2：数据局已同步但组织名称与预期不同
解决：在组织列表中按关键词搜索，确认同步过来的实际名称


6.5 用户列表中找不到目标人员

原因1：该用户尚未由数据局同步到本系统
解决：确认该用户已在政务云平台注册，且所属单位已开通数据同步权限

原因2：数据局已同步但用户信息有误
解决：联系数据局核对同步数据


七、操作检查清单
----------------

7.1 新增单位前检查

□ 确认该单位已在数据局数据访问范围内（对照 1.3 节列表）
□ 如不在列表中，已联系数据局申请开通
□ 数据局已确认完成开通配置
□ 该单位的组织已出现在管理后台组织列表中


7.2 配置权限后检查

□ 组织的工程类型和作业场所已正确配置
□ 用户的所属组织、角色、权限模式已正确配置
□ 如为自定义模式，自定义的工程类型和作业场所已正确配置
□ 需要开通入口权限的人员名单已整理并提交数据局


7.3 开通完成后验证

□ 用户能在政务云首页看到小型工程系统入口图标
□ 用户点击进入系统后能看到备案数据
□ 用户能看到的数据范围与其角色和权限配置一致
