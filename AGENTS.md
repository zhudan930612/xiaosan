# Agent Skills 索引

本项目使用的 Agent Skills 列表。

## 核心工作流技能

| 技能 | 触发词 | 说明 |
|------|--------|------|
| write-feature-doc | 编写功能文档、记录功能、文档化功能 | 编写功能描述文档，输出到 `02_已上线功能描述/` |
| feature-design | 我要做XX功能、优化XX功能、设计XX功能 | 功能设计方案输出，输出到 `03_需求设计文件/` |
| prototype-design | 生成原型、做原型设计、页面原型 | HTML 原型设计，输出到 `04_原型设计文件/` |
| doc-sync | /sync、同步文档 | 文档同步更新，保持产品上下文和规划的一致性 |
| planning-advice | 下一步做什么、功能优先级、规划建议 | 基于产品规划输出功能优先级建议 |

## 辅助技能

| 技能 | 触发词 | 说明 |
|------|--------|------|
| diff-analysis | 对比、区别、差异、核对、清单对照 | 严格按流程对比两份文档或清单的差异 |
| doc-maintenance | 文档约束、格式规范、文档规则 | 文档维护约束和格式规范查询 |
| experience-triage | 经验分诊、该写到哪、新流程放哪 | 判断经验该沉淀到 agent 架构的哪一层 |

## 快速参考

### 文档定位
- 产品上下文：`01_产品主文档/*产品上下文.md`
- 产品规划：`01_产品主文档/*产品规划.md`
- 功能描述：`02_已上线功能描述/*模块.md`
- 功能设计：`03_需求设计文件/*功能设计.md`
- 原型设计：`04_原型设计文件/*.html`
- 决策记录：`05_决策记录文件/*决策记录.md`

### 文件编号
- `03/` 和 `04/` 下使用两位数字前缀 `NN_`
- 功能设计与原型设计编号必须一致
