# doc-sync 技能

## 简介
文档同步技能用于在功能文档完成后,自动同步更新产品上下文和产品规划,确保文档体系的一致性。

## 使用场景
- 功能文档编写完成后
- 需要更新产品上下文中的"已实现功能"
- 需要更新产品规划中的功能状态

## 触发方式
- 说"同步文档"
- 说"更新产品上下文"
- 说"更新产品规划"
- 使用命令 `/sync`

## 执行流程
1. 读取刚完成的功能文档
2. 读取产品上下文和产品规划
3. 在产品上下文中添加新功能
4. 在产品规划中更新功能状态
5. 向用户确认更新内容
6. 执行文档更新

## 注意事项
- 只添加新内容,不修改已有内容
- 保持文档格式和结构一致性
- 更新后建议提交git commit
