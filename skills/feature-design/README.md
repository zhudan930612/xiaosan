# 功能设计技能

基于产品上下文和功能设计模板,输出完整的功能设计方案。

## 触发方式

- 自然语言: "设计XX功能"、"我要做XX功能"、"优化XX功能"
- 命令式: `/design`

## 功能特点

1. **自动读取背景资料**: 自动读取产品上下文、产品规划、功能描述文档
2. **结构化输出**: 使用功能设计模板的结构,输出完整的设计方案
3. **多方案对比**: 原型设计提供3个方案,帮助PM做出更好的决策
4. **明确合并路径**: 提醒用户将"功能点描述"合并到目标模块文档

## 输出内容

- 文档信息
- 功能概述
- 业务流程 (流程图 + 业务规则)
- 原型设计 (3个方案 + 推荐理由)
- 功能点描述 (入口路径、页面功能、业务规则、待补充内容)
- 验收标准

## 使用示例

```
用户: 设计备案列表的筛选功能
AI: [自动读取产品上下文、产品规划、备案管理模块文档]
AI: [输出完整的功能设计方案]
```
