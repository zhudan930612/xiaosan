# 规划建议技能

基于产品规划和产品上下文,提供功能优先级建议。

## 触发方式

- 自然语言: "下一步应该做什么功能"、"应该做什么功能"、"优先级建议"
- 命令式: `/plan-advice`

## 功能特点

1. **自动读取规划文档**: 自动读取产品规划、产品上下文、功能描述文档
2. **基于战略原则**: 严格遵守产品上下文中的功能规划原则
3. **结构化输出**: 按优先级分类输出建议(高/中/低)
4. **提供决策依据**: 明确引用产品规划和产品上下文中的依据

## 输出内容

- 高优先级功能 (建议立即启动)
- 中优先级功能 (建议近期规划)
- 低优先级功能 (建议延后考虑)
- 待解决问题建议
- 已上线功能优化建议

## 分析维度

1. **政策合规** (最高优先级)
2. **用户高频需求** (优先实现)
3. **商业价值驱动** (重点打磨)
4. **技术创新** (适度投入)

## 使用示例

```
用户: 下一步应该做什么功能?
AI: [自动读取产品规划、产品上下文、功能描述文档]
AI: [输出结构化的优先级建议]
```
