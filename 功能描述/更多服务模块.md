# 更多服务模块

## 模块概述

**模块名称**: 更多服务
**模块定位**: 施工端个人中心功能集合，为用户提供帮助文档、客服联系、平台信息查看等辅助功能
**使用角色**: 业主/建设单位、施工单位
**入口位置**: 施工端底部导航栏"更多服务"Tab

---

## 页面结构

### 1. 更多服务首页

**页面布局**:
- **顶部区域**: 用户头像 + 用户名/手机号
- **帮助中心** 功能分组:
  - 备案指南（备案流程与材料）
  - 常见问题（备案常见问题解答）
  - 联系客服（监管部门咨询电话）
- **关于我们** 功能分组:
  - 服务协议
  - 隐私政策

**交互说明**:
- 点击功能项进入对应页面
- 右侧箭头指示可进入下级页面

## 功能详情

### 2. 备案指南

**功能说明**: 帮助用户了解备案流程和准备材料，降低备案申报的驳回率

**页面内容**:
1. **备案流程图**
   ```
   注册登录 → 选择工程类型 → 填写基础信息 → 填写工程信息 → 填写安全信息 → 匹配保单 → 提交审核 → 人工审核 → 获取回执
   ```

2. **所需材料清单**

   **基础信息阶段**:
   - 所有权证明或相关产权证明（必传）
   - 租赁合同（承租人需提供）
   - 建设单位营业执照（机构需提供，支持OCR识别）
   - 甲方负责人身份证（必传）

   **工程信息阶段**:
   - 施工负责人身份证（必传，支持OCR识别）
   - 承发包合同（必传，需提交封面、工程地址、金额、签字盖章页）
   - 施工图纸及技术资料（必传）
   - 相关项目批复/认定手续（必传）
   - 安全风险告知书（需阅读确认）
   - 安全生产承诺书（需阅读并签字）

   **安全信息阶段**（根据作业类型而定）:
   - 施工作业方案/施工专项方案（必传）
   - 施工人员身份证及特种作业证（根据工种而定）
   - 其他材料（如：涉路施工许可证、地下管道交底记录、批准文件等）

   **匹配保单阶段**:
   - 保单文件（上传保单）

3. **材料模板下载**
   - 承发包合同模板.docx
   - 施工方案模板.docx
   - 施工专项方案模板.docx
   - 安全风险告知书.pdf
   - 安全生产承诺书.pdf

4. **注意事项**
   - 材料必须清晰完整，支持JPG、PNG、PDF格式
   - 施工单位必须已认证（可在备案时新增施工单位）
   - 工程地址必须准确（小型工程和零星作业支持地图选点，住宅装修需选择楼栋）
   - 工程总额和建筑面积必须真实，需符合工程规模要求
   - 特种作业证要求：电工、电焊工、高空作业工必须上传
   - 安全生产承诺书需手写签名确认
   - 保单匹配是必要环节，必须完成才能提交

**交互说明**:
- 点击材料模板可下载文件
- 支持长按保存图片（流程图）

### 3. 常见问题

**功能说明**: 整理高频问题解答，减少客服咨询压力，提升用户自助解决问题的能力

**问题分类**:
- 全部
- 账号登录
- 备案申报
- 企业认证
- 材料上传
- 审核流程

**问题列表**:

**账号与登录类**:

Q1: 如何注册账号？
A: 本系统对接青浦区统一认证系统，使用手机号验证码登录即可，无需单独注册。

Q2: 忘记密码怎么办？
A: 本系统使用手机号验证码登录，无需密码。如手机号无法使用，请联系客服。

Q3: 可以更换手机号吗？
A: 可以。请联系客服提供身份证明材料，客服审核后可更换手机号。

**备案申报类**:

Q4: 什么工程需要备案？
A: 青浦区范围内的小型工程、住宅装修、零星作业都需要备案。具体标准请查看备案指南。

Q5: 备案需要多长时间？
A: 提交备案后，街镇审核人员会在1-3个工作日内完成审核。

Q6: 备案被驳回怎么办？
A: 查看驳回原因，修改相关信息后重新提交即可。

Q7: 备案通过后可以修改信息吗？
A: 备案通过后不可修改。如需修改，请联系街镇审核人员。

**企业认证类**:

Q8: 企业认证需要多长时间？
A: 提交认证申请后，审核人员会在1-3个工作日内完成审核。

Q9: 企业认证被驳回怎么办？
A: 查看驳回原因，补充或修改相关材料后重新提交即可。

Q10: 一个账号可以认证多个企业吗？
A: 可以。一个账号可以认证多个企业，并可在不同企业间切换。

**材料上传类**:

Q11: 支持哪些文件格式？
A: 支持图片格式（JPG、PNG）和PDF格式。单个文件不超过10MB。

Q12: 材料上传失败怎么办？
A: 请检查文件格式和大小是否符合要求。如仍失败，请联系客服。

Q13: 材料不清晰怎么办？
A: 请重新拍照上传，确保文字清晰可见。

**审核流程类**:

Q14: 如何查看审核进度？
A: 在"备案列表"中可查看审核状态。审核通过或驳回后会收到短信通知。

Q15: 审核通过后如何获取回执？
A: 审核通过后，在备案详情页面可查看和下载回执。

**交互说明**:
- 点击问题标题展开/收起答案
- 默认全部收起
- 支持按分类筛选问题

### 颜色规范
- 主色: #52c41a (绿色)

- 文字主色: #333
- 文字次色: #999
- 背景色: #f5f7fa
- 分组标题背景: #f0f0f0
- 边框色: #e8e8e8

- 使用统一的图标库（Ant Design Icons）
- 功能图标大小: 24px
- 图标颜色: #52c41a（主色）

- 功能项内边距: 16px
- 分组标题内边距: 12px 16px
