## 一、功能背景

### 1.1 业务痛点

当前施工端用户在使用系统时存在以下问题:

1. **缺少统一的功能入口**:企业认证、帮助文档、客服联系等功能分散,用户找不到入口
2. **备案流程不清晰**:施工方和业主对备案流程、所需材料不了解,导致备案申报时材料不齐全、反复驳回
3. **问题无处咨询**:用户遇到问题不知道如何联系客服或查看帮助文档
4. **缺少服务协议和隐私政策入口**:用户无法查看平台的服务协议和隐私政策

### 1.2 解决方案

通过"个人中心"功能,实现:

1. **统一功能入口**:将企业认证、帮助文档、客服联系等功能集中在个人中心
2. **备案指南**:提供分角色(业主/施工方)的备案流程图文说明和材料清单模板下载
3. **常见问题**:整理高频问题解答,帮助用户自助解决问题
4. **联系客服**:提供监管部门咨询电话和平台客服电话
5. **关于我们**:展示服务协议、隐私政策等平台信息
6. **退出登录**:提供安全的退出登录功能

## 二、核心概念

### 2.1 个人中心定位

个人中心是施工端用户的功能集合页面,包含:
- 企业认证入口(已在"施工单位入驻认证功能设计.md"中详细说明)
- 备案指南(帮助用户了解备案流程和所需材料)
- 常见问题(高频问题解答)
- 联系客服(监管部门和平台客服电话)
- 关于我们(服务协议、隐私政策、平台介绍)
- 退出登录

### 2.2 备案指南

**定位**:帮助用户快速了解备案流程和准备材料,降低备案申报的驳回率

**核心内容**:
- 分角色流程说明(业主/施工方)
- 材料清单和模板下载
- 注意事项提醒

### 2.3 常见问题

**定位**:整理高频问题解答,减少客服咨询压力,提升用户自助解决问题的能力

**问题分类**:
- 账号与登录类
- 备案申报类
- 企业认证类
- 材料上传类
- 审核流程类
- 其他问题

### 3.2 备案指南

#### 3.2.1 页面结构

**Tab切换**:
- 业主备案指南
- 施工方备案指南

**业主备案指南内容**:

1. **备案流程图**
   ```
   注册登录 → 选择工程类型(小型工程/住宅装修/零星作业) → 填写基础信息 → 填写工程信息 → 填写安全信息 → 匹配保单 → 提交审核 → AI审核(可选) → 人工审核 → 获取回执
   ```

2. **所需材料清单**

   **基础信息阶段**:
   - 所有权证明或相关产权证明(必传)
   - 租赁合同(承租人需提供)
   - 建设单位营业执照(机构需提供,支持OCR识别)
   - 甲方负责人身份证(必传)

   **工程信息阶段**:
   - 施工负责人身份证(必传,支持OCR识别)
   - 承发包合同(必传,需提交封面、工程地址、金额、签字盖章页)
   - 施工图纸及技术资料(必传)
   - 相关项目批复/认定手续(必传)
   - 安全风险告知书(需阅读确认)
   - 安全生产承诺书(需阅读并签字)

   **安全信息阶段**(根据作业类型而定):
   - 施工作业方案/施工专项方案(必传)
   - 施工人员身份证及特种作业证(根据工种而定)
   - 其他材料(如:涉路施工许可证、地下管道交底记录、批准文件等)

   **匹配保单阶段**:
   - 保单文件(线上投保或上传保单二选一)

3. **材料模板下载**
   - 承发包合同模板.docx
   - 施工方案模板.docx
   - 施工专项方案模板.docx
   - 安全风险告知书.pdf
   - 安全生产承诺书.pdf

4. **注意事项**
   - 材料必须清晰完整,支持JPG、PNG、PDF格式
   - 施工单位必须已认证(可在备案时新增施工单位)
   - 工程地址必须准确(小型工程和零星作业支持地图选点,住宅装修需选择楼栋)
   - 工程总额和建筑面积必须真实,需符合工程规模要求
   - 特种作业证要求:电工、电焊工、高空作业工必须上传
   - 安全生产承诺书需手写签名确认
   - 保单匹配是必要环节,必须完成才能提交

**施工方备案指南内容**:

1. **备案流程图**
   ```
   企业认证 → 注册登录 → 选择工程类型(小型工程/住宅装修/零星作业) → 填写基础信息 → 填写工程信息 → 填写安全信息 → 匹配保单 → 提交审核 → AI审核(可选) → 人工审核 → 获取回执
   ```

2. **所需材料清单**

   **企业认证阶段**(首次使用需完成):
   - 施工单位营业执照(必传,支持OCR识别)
   - 资质证明(必传)
   - 安全生产许可证(必传)
   - 法人身份证(必传,正反面+手持身份证)

   **基础信息阶段**:
   - 所有权证明或相关产权证明(必传)
   - 租赁合同(承租人需提供)
   - 建设单位营业执照(机构需提供)
   - 甲方负责人身份证(必传)

   **工程信息阶段**:
   - 施工负责人身份证(必传,支持OCR识别)
   - 承发包合同(必传,需提交封面、工程地址、金额、签字盖章页)
   - 施工图纸及技术资料(必传)
   - 相关项目批复/认定手续(必传)
   - 安全风险告知书(需阅读确认)
   - 安全生产承诺书(需阅读并签字)

   **安全信息阶段**(根据作业类型而定):
   - 施工作业方案/施工专项方案(必传)
   - 施工人员身份证及特种作业证(根据工种而定)
   - 其他材料(如:涉路施工许可证、地下管道交底记录、批准文件等)

   **匹配保单阶段**:
   - 保单文件(线上投保或上传保单二选一)

3. **材料模板下载**
   - 承发包合同模板.docx
   - 施工方案模板.docx
   - 施工专项方案模板.docx
   - 安全风险告知书.pdf
   - 安全生产承诺书.pdf

4. **注意事项**
   - 企业必须先完成认证,认证审核需1-3个工作日
   - 材料必须清晰完整,支持JPG、PNG、PDF格式
   - 工程地址必须准确(小型工程和零星作业支持地图选点,住宅装修需选择楼栋)
   - 工程总额和建筑面积必须真实,需符合工程规模要求
   - 特种作业证要求:电工、电焊工、高空作业工必须上传,且必须在有效期内
   - 安全生产承诺书需手写签名确认
   - 保单匹配是必要环节,必须完成才能提交
   - 施工作业风险类别共17种,根据实际情况选择
   - 审核预计1-3个工作日完成,AI审核(如启用)需30-60秒

#### 3.2.2 交互说明

- Tab切换时内容区域切换

**问题列表**:

- 问题答案(展开后显示)

#### 3.3.2 常见问题内容

**账号与登录类**:

Q1: 如何注册账号?
A: 本系统对接青浦区统一认证系统,使用手机号验证码登录即可,无需单独注册。

Q2: 忘记密码怎么办?
A: 本系统使用手机号验证码登录,无需密码。如手机号无法使用,请联系客服。

Q3: 可以更换手机号吗?
A: 可以。请联系客服提供身份证明材料,客服审核后可更换手机号。

**备案申报类**:

Q4: 什么工程需要备案?
A: 青浦区范围内的小型工程、住宅装修、零星作业都需要备案。具体标准请查看备案指南。

Q5: 备案需要多长时间?
A: 提交备案后,街镇审核人员会在1-3个工作日内完成审核。

Q6: 备案被驳回怎么办?
A: 查看驳回原因,修改相关信息后重新提交即可。

Q7: 备案通过后可以修改信息吗?
A: 备案通过后不可修改。如需修改,请联系街镇审核人员。

**企业认证类**:

Q8: 企业认证需要多长时间?
A: 提交认证申请后,审核人员会在1-3个工作日内完成审核。

Q9: 企业认证被驳回怎么办?
A: 查看驳回原因,补充或修改相关材料后重新提交即可。

Q10: 一个账号可以认证多个企业吗?
A: 可以。一个账号可以认证多个企业,并可在不同企业间切换。

**材料上传类**:

Q11: 支持哪些文件格式?
A: 支持图片格式(JPG、PNG)和PDF格式。单个文件不超过10MB。

Q12: 材料上传失败怎么办?
A: 请检查文件格式和大小是否符合要求。如仍失败,请联系客服。

Q13: 材料不清晰怎么办?
A: 请重新拍照上传,确保文字清晰可见。

**审核流程类**:

Q14: 如何查看审核进度?
A: 在"备案列表"中可查看审核状态。审核通过或驳回后会收到短信通知。

Q15: 审核通过后如何获取回执?
A: 审核通过后,在备案详情页面可查看和下载回执。
