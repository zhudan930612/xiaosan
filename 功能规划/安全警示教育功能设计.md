## 功能单元 1：安全警示教育页面

> 💡 **合并说明**: 本功能单元完成后,将"功能点描述"章节合并到 [备案管理模块.md] 中。

### 功能概述

在施工人员选择备案类型后、进入资料填写页面前，强制展示安全警示教育内容。页面通过视频置顶自动播放+文案说明的方式，向用户传达安全施工的重要性、常见隐患风险和防范措施。用户滑动浏览完内容后点击"我已知晓"进入下一步，同时支持"近30天内不再显示"选项，减少高频备案用户的重复操作。

---

### 决策记录

1.  decision: 所有备案类型共用同一视频还是各配独立视频  choice: 所有备案类型共用同一视频（时长4分40秒，约300MB）  basis: 客户确认视频内容为通用安全教育，不区分备案类型（基于用户访谈确认）
2.  decision: 30天免显示状态存储位置  choice: 存储在设备本地，换设备需重新确认  basis: 产品规划已明确能力边界，且为纯用户体验优化功能，不涉及关键数据（基于产品规划）
3.  decision: 视频是否要求用户完整观看  choice: 不强制看完，进入页面后自动播放，用户可随时滑动或点击"我已知晓"跳过  basis: 提升用户体验，避免强制性操作导致用户流失（基于用户访谈确认）
4.  decision: 是否由服务端记录用户确认行为  choice: 仅本地存储，服务端暂不做记录  basis: 当前产品规划定位为纯本地功能，如需统计分析后续可扩展（基于产品规划）
5.  decision: "近30天内不再显示"复选框默认状态  choice: 默认未选中  basis: 确保用户主动选择免显示，避免默认跳过安全教育（基于设计推断）⚠️ 推测，待确认
6.  decision: 页面文案是否配文字版视频要点  choice: 视频下方采用"核心要点摘要 + 分段展开说明 + 完整文案收起"的组合结构，文案内容与视频实际口径保持一致，不再单独拆成通用的五类施工须知  basis: 便于用户先快速获取重点，再按需查看细节，同时避免页面文案与当前视频内容不一致（基于用户访谈确认 + 基于视频文案提取结果）

**流程补充说明**（基于用户访谈确认）：
三种备案类型（小型工程/住宅装修/零星作业）各自独立计算30天免显示周期。本地存储使用设备本地缓存，换设备需重新确认。视频播放不要求看完，用户可随时滑动查看下方文案。视频文件时长4分40秒（约280秒），大小300MB+，所有备案类型共用同一视频。不记录用户行为数据，纯本地功能。

#### 业务规则

触发条件（基于用户访谈确认）：
用户在"选择工程类型"页面点击任意工程类型卡片（小型工程备案/住宅装修备案/零星作业备案）后，系统检查本地存储，若该类型最后一次确认时间超过30天或从未确认，则进入安全警示教育页面。

处理逻辑（基于产品规划及用户访谈确认）：
1. 用户点击备案类型卡片。
2. 系统读取本地存储，获取该类型最后一次确认时间戳。
3. 判断是否需要显示教育页面：未找到记录则显示；找到记录且距今天数不超过30天则跳过，直接进入备案资料填写；找到记录且距今天数超过30天则显示。
4. 进入教育页面后视频自动播放（静音循环播放）。
5. 用户滑动页面浏览文案内容。
6. 滑动至底部显示操作区，包括"我已知晓"按钮和"近30天内不再显示"选项。
7. 用户点击"我已知晓"按钮：若勾选了"近30天内不再显示"，记录当前时间戳到本地存储，随后进入备案资料填写页面。

边界情况（基于设计推断，部分⚠️ 待确认）：
- 视频加载失败：显示视频占位图（带重试按钮），文案内容仍可正常浏览。⚠️ 推测，待确认
- 网络中断：页面已缓存时可正常展示，未缓存时提示网络异常。⚠️ 推测，待确认
- 用户快速滑动：不限制滑动速度，但"我已知晓"按钮固定于底部显示。
- 未勾选免显示选项：每次点击备案类型都进入教育页面。
- 清除缓存：本地存储被清除后，下次需重新确认。

#### 安全警示教育页面

**入口路径**（基于[备案管理模块.md]）：
施工端App → 项目中心 → 点击"工程备案"快捷入口 或 "+"按钮 → 选择工程类型 → 进入安全警示教育页面（若需显示）。

**页面名称**："安全警示教育"页面

**页面功能**（基于用户访谈确认，部分基于设计推断）：

1. 视频播放区：视频置顶显示，占据屏幕上方约60%区域，采用16:9比例并自适应宽度。视频自动播放、静音、循环播放。点击视频可进入全屏模式。视频加载失败时显示占位图和重试按钮。⚠️ 占位图及重试机制为推测，待确认。

2. 滑动提示区：视频下方显示滑动引导文字，采用呼吸灯动画效果（上下浮动箭头），用户开始滑动后自动消失。

3. 文案内容区：当前三种备案类型共用同一视频和同一套文案，页面标题统一为"小型工程安全警示要点"。视频下方的文案按三层结构展示：
   - 第一层为"核心提示摘要"，展示4-6条短句要点，帮助用户在短时间内理解本页重点，包括小型工程风险提醒、责任主体要求、保险投保提醒、四类高风险作业提示等内容。
   - 第二层为"分段展开说明"，按视频实际结构拆分为5个模块：为什么要看这段安全教育、相关责任主体要求、保险投保提醒、四类高风险作业场景、结语与安全倡议。默认展示模块标题和摘要，支持点击展开查看详细内容。
   - 第三层为"完整文案"，默认收起，用户点击"展开完整文案"后查看视频对应的连续文字版内容，用于补充阅读，不作为首屏默认展示内容。
   后续如客户改为不同备案类型配置不同视频和文案，仍保留按类型独立配置标题、摘要、分段内容和完整文案的能力边界。

4. 底部操作区（固定定位）：包含复选框"近30天内不再显示此页面"和主按钮"我已知晓"（绿色填充按钮样式，基于[备案管理模块.md]第209-223行）。

**业务规则**（基于用户访谈及产品规划确认）：
三种备案类型（小型工程/住宅装修/零星作业）各自独立计算30天免显示周期。30天计算逻辑为从最后一次点击"我已知晓"并勾选免显示选项的时刻开始计算。本地存储键名格式为 `safety_education_[工程类型代码]_confirmed_at`。视频播放不强制看完，用户可随时点击"我已知晓"进入下一步。未勾选免显示选项时，每次选择该备案类型都重新进入教育页面。当前所有备案类型共用同一视频和同一套文案内容。文案展示顺序固定为"摘要要点 → 分段展开内容 → 完整文案入口"，其中完整文案默认收起。点击返回按钮不记录确认状态，下次进入仍需重新确认。本功能不记录用户行为数据，属于纯本地功能。

**视频信息**（基于用户访谈确认）：
视频时长4分40秒（约280秒），文件大小约300MB，所有备案类型共用同一视频。视频由客户确认，内容结构如下：
一、开篇
定义小型工程，并强调施工风险问题。
二、责任提醒
1. 落实参建主体责任：包括业主责任、建设单位责任、施工单位责任。
三、保险提醒
推进安责险投保，并强调开工前完成投保、保单覆盖完整施工周期。
四、高风险作业提醒
四类事故高发作业场景的安全防范提醒：高处作业、临时用电、人工拆除、动火作业。
五、结语
强调"小工程，大责任"，落实安全责任，共筑安全防线。

3. 页面文案是否需要支持后台独立配置摘要内容、分段文案和完整文案。⚠️ 待确认
