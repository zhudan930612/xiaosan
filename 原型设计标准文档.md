# 原型设计标准文档

> 本文档定义小型工程安全纳管系统的UI设计规范，用于指导AI生成功能设计文档中的原型方案。
> 所有原型必须遵循本规范，保持系统视觉一致性。

---

## 1. 设计系统概述

### 1.1 设计原则

- **清晰层级**：通过卡片、间距、色彩区分信息层级
- **流程可视化**：使用流程状态卡片展示业务进度
- **分组明确**：相关内容使用卡片聚合，标题清晰标识
- **操作突出**：主操作按钮使用品牌色，位置固定

### 1.2 应用场景

| 场景类型 | 应用端 | 设计特点 |
|---------|--------|---------|
| 备案申请 | 微信小程序（业主/施工方） | 表单为主，分步骤填写 |
| 巡查检查 | 移动端APP（巡查人员） | 拍照上传，位置定位 |
| 管理后台 | PC Web（管理人员） | 表格列表，数据统计 |

## 2. 色彩规范

### 2.1 主色调

| 色彩角色 | 色值 | 使用场景 |
|---------|------|---------|
| **品牌主色** | `#00B578` | 主按钮、选中状态、重要标识 |
| **品牌辅色** | `#00A86B` | 渐变结束色、强调元素 |
| **品牌浅绿** | `#E6F7F0` | 浅色背景、标签背景 |

### 2.2 渐变色

**顶部导航渐变**：
```
起始色：#00B578
结束色：#00A86B
方向：从上到下或从左到右
```

**状态卡片渐变**（流程进度卡片）：
```
起始色：#00B578
结束色：#00A86B
方向：135度对角渐变
```

### 2.3 功能色

| 色彩角色 | 色值 | 使用场景 |
|---------|------|---------|
| 成功/通过 | `#00B578` | 审核通过、正常状态、已整改 |
| 警告/待办 | `#FF9500` | 待审核、待处理、待整改 |
| 错误/拒绝 | `#FF3B30` | 审核不通过、异常状态、已驳回 |
| 信息/默认 | `#8E8E93` | 未开始、默认状态 |
| **风险/高危** | `#FF4D4F` | 高风险项目、红色标签 |
| **一般风险** | `#FAAD14` | 一般风险、黄色标签 |
| **处理中** | `#1890FF` | 待受理、蓝色标签 |
| **灰色状态** | `#999999` | 已整改、灰色标签 |

### 2.4 文字色

| 色彩角色 | 色值 | 使用场景 |
|---------|------|---------|
| 主文字 | `#333333` | 标题、重要内容 |
| 次要文字 | `#666666` | 正文、描述 |
| 辅助文字 | `#999999` | 提示、说明 |
| 链接文字 | `#00B578` | 可点击链接 |

### 2.5 背景色

| 色彩角色 | 色值 | 使用场景 |
|---------|------|---------|
| 页面背景 | `#F5F5F5` | 页面底色 |
| 卡片背景 | `#FFFFFF` | 内容卡片 |
| 分割线 | `#E5E5E5` | 分隔线 |

## 3. 布局规范

### 3.1 页面结构

```
┌─────────────────────────────┐
│ 状态栏（时间、信号、电量）      │  ← 系统自带
├─────────────────────────────┤
│  ← 页面标题              ... │  ← 导航栏（固定高度44px）
├─────────────────────────────┤
│                             │
│      流程状态卡片（如有）      │  ← 业务流程展示
│                             │
├─────────────────────────────┤
│  标签1  │ 标签2  │ 标签3     │  ← 标签导航（多步骤表单）
├─────────────────────────────┤
│                             │
│   ▎分组标题                 │  ← 信息分组标识
│   ┌─────────────────────┐  │
│   │                     │  │
│   │    内容卡片          │  │  ← 信息展示区
│   │                     │  │
│   └─────────────────────┘  │
│                             │
│   ▎分组标题                 │
│   ┌─────────────────────┐  │
│   │                     │  │
│   │    内容卡片          │  │
│   │                     │  │
│   └─────────────────────┘  │
│                             │
├─────────────────────────────┤
│  [辅助按钮]    [主操作按钮]   │  ← 底部操作区（固定）
└─────────────────────────────┘
```

### 3.2 间距规范

| 元素 | 间距 | 说明 |
|-----|------|-----|
| 页面边距 | 16px | 左右边距 |
| 卡片内边距 | 16px | 卡片内部间距 |
| 卡片间距 | 12px | 卡片之间间距 |
| 分组间距 | 24px | 不同分组之间间距 |
| 行间距 | 12px | 表单项之间间距 |
| 标签栏高度 | 48px | 标签导航高度 |
| 底部按钮区 | 80px | 底部固定操作区高度 |

### 3.3 圆角规范

| 元素 | 圆角 | 说明 |
|-----|------|-----|
| 大卡片 | 12px | 主要内容卡片 |
| 小卡片/标签 | 8px | 小模块、标签 |
| 按钮 | 24px | 胶囊形圆角（高度的一半） |
| 输入框 | 8px | 表单输入框 |
| 图标容器 | 8px | 图标背景容器 |

## 4. 组件规范

### 4.1 导航栏

**标准导航栏**：
- 背景：渐变 `#00B578` → `#00A86B`
- 高度：44px + 系统状态栏
- 标题：居中，白色，18px，加粗
- 返回按钮：左侧，白色箭头图标
- 右侧操作：可选，白色图标

```
┌─────────────────────────────┐
│  ←        页面标题        ... │
└─────────────────────────────┘
  白          白              白
```

### 4.2 流程状态卡片

**适用场景**：展示业务流程进度（如备案申请流程）

**样式规范**：
- 背景：渐变色 `#00B578` → `#00A86B`
- 圆角：12px
- 内边距：20px
- 内容：左右分栏展示当前状态和下一状态

```
┌─────────────────────────────────────┐
│                                     │
│   [图标]         >>>>>     [图标]    │
│                                     │
│   当前状态               下一状态    │
│   时间                  操作按钮    │
│                            │        │
└────────────────────────────┼────────┘
                             │
                        悬浮标签
```

**右上角悬浮标签**：
- 背景：白色半透明
- 圆角：左下角12px
- 文字：品牌绿色 `#00B578`
- 图标：文档图标

### 4.3 标签导航（Tab）

**适用场景**：多步骤表单、多维度信息切换

**样式规范**：
- 背景：白色
- 高度：48px
- 文字：14px
- 默认状态：文字 `#666666`
- 选中状态：文字 `#00B578`，底部绿色下划线（3px）

```
┌─────────────────────────────────────┐
│   标签1    │   标签2   │   标签3    │
│   ─────                ─────        │
│    选中                选中         │
└─────────────────────────────────────┘
```

### 4.4 信息分组卡片

**分组标题**：
- 前缀：绿色竖条（4px宽，16px高）`#00B578`
- 文字：16px，加粗，`#333333`
- 左侧间距：16px

**内容卡片**：
- 背景：白色
- 圆角：12px
- 内边距：16px
- 阴影：无（扁平化设计）

**表单项布局**：
```
┌─────────────────────────────┐
│                             │
│  标签              值       │
│  #666666          #333333   │
│  左对齐           右对齐     │
│                             │
│  ─────────────────────────  │  ← 分割线
│                             │
│  标签              值       │
│                             │
└─────────────────────────────┘
```

### 4.5 按钮规范

**主按钮（Primary）**：
- 背景：渐变色 `#00B578` → `#00A86B`
- 文字：白色，16px，居中
- 高度：48px
- 圆角：24px（全圆角胶囊形）
- 位置：底部固定或卡片内部

**次按钮（Secondary）**：
- 背景：白色
- 边框：1px solid `#00B578`
- 文字：`#00B578`，16px
- 高度：48px
- 圆角：24px

**底部双按钮布局**：
```
┌─────────────────────────────────────┐
│                                     │
│  [    次按钮    ]  [    主按钮    ]  │
│   白色底绿字         渐变绿底白字     │
│      48%              48%          │
│                                     │
└─────────────────────────────────────┘
```

### 4.6 信息提示卡片

**政策说明卡片**：
- 背景：`#E6F7F0`（浅绿）
- 边框：1px solid `#00B578`
- 圆角：12px
- 标题栏：渐变背景，白色文字，左侧图标
- 内容区：深色文字，正文大小

```
┌─────────────────────────────┐
│ [图标] 标题           [>>]  │  ← 渐变绿底标题栏
├─────────────────────────────┤
│                             │
│   政策说明内容正文...        │  ← 浅绿背景内容区
│                             │
│   [补充说明文字框]           │
│                             │
└─────────────────────────────┘
```

**上传按钮**：
- 尺寸：100px × 100px
- 边框：1px dashed `#00B578`（虚线）
- 圆角：8px
- 图标：上传图标，`#00B578`
- 文字：`#00B578`，12px

**已上传文件**：

### 4.8 底部信息栏

**客服信息栏**：
- 背景：白色
- 左侧：图标（圆形头像/图标）
- 中间：文字说明
- 右侧：操作按钮（绿色边框胶囊按钮）

```
┌─────────────────────────────────────┐
│                                     │
│  [图标]  说明文字         [按钮]     │
│          灰色              绿边框   │
```

## 5. 表单组件规范

### 5.1 单选框（Radio）

**样式规范**：
- 未选中：空心圆，边框 `#CCCCCC`，20px
- 选中：实心圆，背景 `#00B578`，内部白色对勾
- 文字：14px，`#333333`，左侧间距8px
- 选项间距：16px

```
○ 未发现        ● 已发现
  灰色圆圈       绿色实心+对勾
```

### 5.2 复选框（Checkbox）

**样式规范**：
- 未选中：方形边框，`#CCCCCC`
- 选中：背景 `#00B578`，白色对勾图标
- 文字：14px，`#333333`
- 纵向排列间距：12px

```
☐ 选项1         ☑ 选项2
  灰框            绿底白勾
```

### 5.3 开关按钮（Switch）

**样式规范**：
- 关闭：灰色背景 `#CCCCCC`，圆形滑块居左
- 开启：绿色背景 `#00B578`，圆形滑块居右
- 尺寸：宽44px，高24px
- 滑块：白色，22px直径

```
    ┌──┐              ┌────┐
    │  │              │    │
    └──┘              │  ┌─┘
   关闭状态            └─┘
                  开启状态
```

### 5.4 标签选择器

**适用场景**：违法行为分类、作业类型选择

**样式规范**：
- 默认状态：白色背景，边框 `#E5E5E5`，文字 `#666666`
- 选中状态：绿色背景 `#00B578`，文字白色
- 圆角：16px（全圆角）
- 内边距：8px 16px
- 布局：流式排列，间距8px

```
┌─────────────────────────────────────┐
│                                     │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐  │
│  │控停 │ │高处 │ │用电 │ │消防 │  │
│  │  1  │ │  2  │ │     │ │     │  │
│  └─────┘ └─────┘ └─────┘ └─────┘  │
│    默认     选中                    │
│                                     │
└─────────────────────────────────────┘
```

### 5.5 搜索栏

**样式规范**：
- 背景：白色
- 左侧：定位图标 + 区域选择文字
- 中间：下拉箭头（选择筛选条件）
- 右侧：输入框 + 搜索按钮
- 高度：48px
- 圆角：24px（全圆角）

```
┌─────────────────────────────────────┐
│  [定位] 南山区 ▼ │ 请选择 ▼ │ 搜索 │
│   图标   区域     筛选条件   搜索按钮│
```

### 5.6 输入框

**样式规范**：
- 背景：白色或 `#F5F5F5`
- 边框：默认无，聚焦时 `#00B578`
- 圆角：8px

```

[信息分组卡片1]
  - 表单项1
  - 表单项2
[信息分组卡片2]
```

**使用说明**：

1. **生成功能设计文档时**，首先读取本文档获取设计规范
2. **根据页面类型选择模板**：
   - 表单填写 → 类型A（备案申请、信息录入）
   - 详情展示 → 类型B（审核结果、工程详情）
   - 列表浏览 → 类型C（工程列表、任务列表）
   - 巡查检查 → 类型D（巡查录入、隐患上报）
   - 人员选择 → 类型E（选择工人、选择巡查员）
3. **严格遵循色彩规范**：
   - 品牌色：`#00B578` → `#00A86B` 渐变
   - 功能色：成功绿、警告橙、错误红、风险红、处理蓝
4. **使用标准组件**：
   - 导航栏：渐变背景+白色标题
   - 流程卡片：渐变背景+右上角悬浮标签
   - 标签导航：选中项绿色下划线
   - 分组标题：绿色竖条前缀
   - 按钮：胶囊形全圆角（主按钮渐变、次按钮白底绿边）
5. **使用ASCII字符绘制原型**，保持结构清晰
6. **在原型后附上色彩标注和交互说明**

**设计原则**：
- 高风险项目必须用**红色标签**（`#FF4D4F`）突出显示
- 状态标签使用**胶囊形状**，颜色对应状态语义
- 工程卡片使用**4列统计网格**展示巡查数据
- 表单页面使用**分组卡片**组织信息，每组配绿色标题前缀

**参考案例**：
- 备案申请页面 → 类型A模板
- 工程列表页面 → 类型C模板
- 巡查录入页面 → 类型D模板
- 人员选择页面 → 类型E模板
